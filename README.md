## このリポジトリはなに？

限界大会公式リポジトリです。  
限界大会のルールやサイトのデータを管理していて、自由に修正・提案することができます。

## ディレクトリ構成

| ディレクトリ                    | 内容                                           |
| ------------------------------- | ---------------------------------------------- |
| [/content/docs](./content/docs) | 各言語ごとのルールやポリシーなどのドキュメント |
| [/i18n](./i18n)                 | 各言語ごとのサイト翻訳ファイル                 |
